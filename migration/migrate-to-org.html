🔄 Migration: Per-User → Org-Struktur

Dieses Skript kopiert alle bestehenden Daten aus dem alten users/{adminUid}/ Pfad in die neue gemeinsame orgs/{orgId}/ Struktur. Führe dies einmalig als Admin aus.

Schritt 1 – Anmelden als Admin

Schritt 2 – Org-ID festlegen

Die Org-ID ist ein eindeutiger Bezeichner für euer Team (z.B. "livetour"). Alle Benutzer müssen dieselbe Org-ID haben.

Schritt 3 – Migration starten

Was passiert:
  • Alle Daten von users/{adminUid}/ werden nach orgs/{orgId}/ kopiert
  • users/{adminUid}/orgId wird gesetzt
  • users/{adminUid}/role wird auf "admin" gesetzt
  • Die alten Daten bleiben als Backup erhalten

Schritt 4 – Weitere User zuweisen

Trage hier die Firebase-UIDs aller anderen Team-Mitglieder ein, um sie dieser Org zuzuweisen.
